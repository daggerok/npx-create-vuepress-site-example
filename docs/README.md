# npx create-vuepress-site

## development

```bash
npm i -E
npm run dev
# open http://127.0.0.1:8080/
```

## testing build

```bash
npm i -E
npm run build
npx serve src/.vuepress/dist
# open http://127.0.0.1:5000/
```

## testing github-pages build

```bash
npm i -E
npm run serve-github-pages
```
